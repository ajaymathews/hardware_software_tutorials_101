# Ajay's Master Tutorial Collection (2023-2026)

Welcome to my comprehensive, multi-year collection of programming and hardware tutorials. This repository preserves the full Git history of 6+ major projects, covering everything from low-level ARM development to high-level Python mini-projects.

## 🚀 Repository Roadmap

### 1. 📟 Microcontrollers
Deep-dives into register-level programming for major architectures:
- **ARM Cortex-M4 (STM32):** GPIO, ADC, PWM, UART, SPI, and RTOS.
- **ATmega328P (AVR):** Peripherals and Interfacing.
- **PIC16F877A:** Core communication and sensor logic.

### 2. 🐧 Linux Development
Professional tutorials for kernel-level programming:
- **Linux Device Drivers:** Module loading, char drivers, `ioctl`, `procfs`, `sysfs`, and Interrupts.

### 3. 💻 Programming Masterclass
A series of concepts and challenges for major languages:
- **C++:** OOP, Inheritance, Polymorphism, and File I/O.
- **Python Basic:** Core syntax, persistence with `pickle`, and File I/O.
- **Shell Scripting:** Automation, Signal handling, and batch operations.

### 4. ⚙️ Hardware & IoT
Practical implementations for single-board computers:
- **Raspberry Pi Sensors:** DHT11, Ultrasonic sensors, Servo control, and ThinkSpeak IoT integration.
