## 🛠 Prerequisites

To run these tutorials, you will need the **GCC (G++) Compiler** installed on your system.

### How to Install G++ (GCC)

- **Windows:**
  - Install **MinGW-w64** via [MSYS2](https://www.msys2.org/) or the standalone MinGW installer.
  - Add the `bin` folder to your system **Path** environment variable.
  - Verify by typing `g++ --version` in your terminal.
- **Linux:**
  - `sudo apt-get update && sudo apt-get install build-essential`
- **macOS:**
  - `xcode-select --install`

## 🚀 How to Compile & Run

Every file in this repository has built-in instructions in its header. Generally, you can use the following commands in your terminal:

### 1. Simple Compilation
```bash
g++ filename.cpp -o program_name
```

### 2. Run the Program
```bash
./program_name
```
