# Shell Scripting: Tutorial

A comprehensive set of 15+ hands-on tutorials for mastering Bash and Shell scripting. This repository covers everything from basic variables and loops to signal handling and Python integration.

## 🚀 Repository Structure

The tutorials are organized into individual scripts, each focused on a specific concept:

- **Basics:** Variable counting, printing, and reading user input.
- **Logic:** Conditionals (if/else) and advanced Loops (for, while, until).
- **Automation:** Batch file renaming and network status checking (IP checker).
- **Core Shell:** Array manipulation, functions, and signal traps.
- **Integration:** Triggering Python scripts from Shell.

## 🏃 Getting Started

### Interactive Tour (Recommended)
I have included an **Interactive Master Script** to help you explore the tutorials easily. It provides a menu system with explanations of each script before running them.

```bash
chmod +x interactive_tour.sh
./interactive_tour.sh
```

### Running Individual Tutorials
You can also run any script directly:
```bash
chmod +x <script_name>.sh
./<script_name>.sh
```

## 🛠 Best Practices Included
- **Quoted Variables:** All scripts follow the best practice of quoting variables to handle spaces and empty values.
- **Standardized Shebangs:** All scripts use `#!/bin/bash` for consistency.
- **Clean Structure:** Temporary and backup files have been removed or gitignored.
